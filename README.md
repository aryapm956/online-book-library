# 📚 Online Book Library

A frontend-based Online Book Library built with Angular that allows users to explore books, search titles, and view detailed book information through API integration. The application features a clean, responsive, and user-friendly interface designed to provide an engaging book browsing experience.

## 🚀 Features

- 📖 Browse a collection of books
- 📚 View detailed book information
- 📱 Responsive user interface
- 🔗 API integration for dynamic book data

## 🛠️ Technologies Used

- Angular 21
- TypeScript
- HTML5
- CSS3
- Bootstrap
- REST API
